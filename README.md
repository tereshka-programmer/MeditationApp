<h1 align="center">Meditation App + Offline music player<h1>

<h2>Stack:<h2>
  
- Android SDK
- Kotlin Coroutines
- Flow
- ExoPlayer
- MediaPlayer
- MVVM
- Glide
- Service
- BroadCast Receiver 
- Single Activity
- Navigation Component
- SafeArgs
- JUnit
