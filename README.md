<h1 align="center">Meditation App + Offline music player

<h2>Stack:
* Android SDK
  - Kotlin Coroutines
  - Flow
  - ExoPlayer
  - MediaPlayer
  - MVVM
  - Glide
  - Service
  - BroadCast Receiver 
  - Single Activity
  - Navigation Component
  - SafeArgs
  - JUnit
